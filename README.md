Tree
===============
Tree component<br>
Display the hierarchical data by tree.<br>
Each node can be moved by drag and drop.

## Feature
* Display hierarchical data by tree UI
* Fold sub tree
* Create & Edit node
* Drag and drop (extend)
* Tri-state checkbox
* Custom events
* Context menu
* Ajax

## Dependency
* tui-code-snippet: ~1.2.0
